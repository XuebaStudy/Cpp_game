# 一个C++文字小游戏 
- 类型：对战小游戏 (类似于**炉石传说**) 
- 多职业、多技能、回合制、多人队伍......
- 相当简陋

## 如何运行游戏 (Windows)
- 运行 Project/bin/main.exe 即可

## 如何快速构建并运行 main.exe (Windows)
1. 尽量保证已有:
    (并已添加到PATH中)(否则容易报错)
    1. MinGW Makefiles
    2. CMake (3.10以上版本)
    3. C++20 支持编译器
2. 进入项目根目录，在PowerShell中执行:`./scripts/main.bat`
